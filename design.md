# SidBuilds Design System

## Aesthetic
Stripe + Linear inspired. Dark, calm, high-end. No clutter, no flashy effects.
Premium founder portfolio feel — surgical precision in every spacing choice.

## Colors
- background: #05070a
- surface: #0b0f14
- elevated: #0f141a
- text: #e6eaf0
- muted: #8b98a5
- border: rgba(255,255,255,0.08)
- accent-blue: #3b82f6
- accent-green: #22c55e
- accent-red: #ef4444

## Typography
- Font: Inter (Google Fonts)
- Hero headings: bold, -0.03em letter-spacing, 1.1 line height
- Section headings: semibold, tight
- Body: muted, 1.7 line height
- Labels: 0.7rem, uppercase, wide tracking

## Spacing Rules
- hero → products: 160px gap
- products → shipping: 100px gap
- shipping → principles: 200px gap
- principles → about: 100px gap

## Motion
- opacity + translateY only
- No bounce, no scale
- Stagger on entrance
- Hover: subtle lift (2-4px translateY)
- Duration: 0.4-0.6s ease-out

## Layout
- Max-width: 1200px
- Padding: 0 1.5rem (mobile) / 0 2rem (desktop)
- Left-aligned hero
- Products in floating slab container
